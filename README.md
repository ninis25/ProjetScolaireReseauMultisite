# 🚀 Projet SAE33 - Documentation et Rapports

Bienvenue dans le dossier de documentation et de rapports du projet SAE33 ! Ce projet a été réalisé par le **Groupe 4**. Vous trouverez ici toutes les informations nécessaires sur les différentes étapes et la progression du projet.

## 📂 Contenu du dossier

### 📌 Jalons
Les jalons représentent les étapes clés de développement et de configuration du projet. Vous trouverez pour chaque jalon des documents détaillés au format DOCX et PDF.

- **📑 Jalon 4** : Première documentation avec analyses initiales et configurations.
  - Fichiers : `Jalon4_Groupe4.docx`, `Jalon4_Groupe4.pdf`
  
- **📝 Jalon 5** : Rapport intermédiaire présentant la progression et l'architecture réseau.
  - Fichiers : `Jalon5_Groupe4.docx`, `Jalon5_Groupe4.pdf`
  
- **🔧 Jalon 6** : Documentation avancée détaillant les configurations et les tests.
  - Fichiers : `Jalon6_Groupe4.docx`, `Jalon6_Groupe4.pdf`
  
- **🌐 Jalon 7** : Configuration GNS3 utilisée pour le projet (réseaux simulés).
  - Fichiers : `Jalon7_groupe4.gns3project`, `Jalon7_groupe4.gns3project.zip`
  
- **📘 Jalon 8** : Rapport final couvrant la phase terminale et les résultats obtenus.
  - Fichiers : `Jalon8_Groupe4.docx`, `Jalon8_Groupe4.pdf`

### 🏆 Rapport Final
Le rapport final regroupe toutes les informations et conclusions du projet, offrant une vue d'ensemble complète sur les activités, configurations, et résultats.

- **Fichiers** : `rapportsae33.docx`, `rapportsae.pdf`

### ⏰ Diagramme de Gantt
Le fichier `Mindview_gantt_groupe4.mvdx` présente le planning et l’échéancier du projet sous forme de diagramme de Gantt.

### 🗂️ Fichiers Temporaires
Des fichiers de configuration et fichiers temporaires sont inclus pour référence. 

## 🔍 Utilisation
Ce répertoire est destiné à fournir une documentation exhaustive pour les utilisateurs et contributeurs du projet SAE33. Chaque jalon est détaillé et les fichiers sont organisés pour faciliter la compréhension et le suivi du projet.
